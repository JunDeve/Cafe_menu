인텔리제이에서 작업을 하였습니다. 상세기능은 README 영상에서 확인 가능합니다.

- 실행방법

인텔리제이를 실행합니다 >
폴더를 가져옵니다 >
터미널(Alt+F12)에서 npm install 입력합니다 >
톰켓서버 실행합니다 >
터미널(Alt+F12)에서 node ./server/server.js 입력합니다 >
주소창에서 http://localhost/ 를 입력합니다 >
사이트에 접속이 완료 됩니다

시간부족 및 코드오류로 인하여, 수정버튼 및 삭제버튼을 프론트에서 구현하였습니다.
아직 백엔드랑 연동을 하지않은상태이므로, 새로고침시 반영이 되지않습니다.
다른 상세기능들은 모두 정상 작동합니다.

처음 사용하는 프로그램이기에 많은 시행착오를 겪었습니다. 하지만 많은 시도끝에
조금이나마 익숙해지고 이해를 하기 시작하며, 추후에 숙련되면 이클립스보다 더 많이 
애용할 것 같은 생각을 하였습니다. 이 프로젝트를 진행하고 완성하면서 코드를 돌아보고
나자신도 돌아보면서, 다방면의 해석능력을 키우고 잘 마무리했다는것에 성취감을 느꼈습니다.
남은 프로젝트 또한 열심히 하겠습니다 이상입니다.
